## 🎯 总体目标

## 📅 学习周期（4 周系统性进阶）
### 第 1 周：Java 基础语法回顾（已完成 ✅）
- 面向对象、异常、泛型  
- 集合框架（ArrayList/HashMap/ConcurrentHashMap）  
- IO、NIO、反射、注解、Lambda & Stream  
- 算法：数组、链表、二叉树、动态规划入门

### 第 2 周：多线程与并发编程（进行中 🚀）
- 线程创建、生命周期、synchronized  
- volatile、JMM、Lock、ReentrantLock  
- 线程池、JUC、CountDownLatch、Semaphore  
- CAS、AQS、ThreadLocal、CompletableFuture  
- 死锁、生产者-消费者模型

### 第 3 周：JVM + MySQL
- JVM 内存结构、对象创建、GC 算法、垃圾收集器  
- 类加载、双亲委派、JVM 参数与工具  
- MySQL 索引 B+、聚簇索引、覆盖索引、最左前缀  
- 事务 ACID、隔离级别、MVCC、行锁/间隙锁  
- SQL 优化、EXPLAIN 分析

### 第 4 周：MySQL 进阶 + 计算机基础 + OS
- redo log/undo log/binlog、慢查询优化  
- MyBatis 原理、缓存、动态SQL  
- 计算机网络：TCP/IP、HTTP/HTTPS、三次握手四次挥手  
- 操作系统：进程/线程、内存、管道、共享内存  
- Linux 常用命令、排查工具

### 收尾冲刺（4.27–4.30）
- 全模块八股复盘 + 口述表达训练  
- 算法高频题复盘 + 独立默写代码
